# 💳 Pricing Panel

A CSS practice project — building a clean, professional three-tier pricing table from scratch using only HTML and CSS.

---

## 📚 What I Was Learning

Building structured, visually polished UI components with pure CSS — specifically pricing table layouts with highlighted "featured" tiers and icon integration.

---

## 🔑 Key Things Practiced

- **CSS Flexbox** — Aligning pricing cards side-by-side and centering content within each card
- **Visual Hierarchy** — Differentiating a "featured" plan from others using color, sizing, and shadow
- **Box Model** — Careful use of padding, borders, and margins to create clean card spacing
- **Icon Integration** — Embedding custom icon images within pricing tiers
- **Starter → Finished Workflow** — The repo includes both a starter template and completed version

---

## 💡 What It Taught Me

Building a pricing table taught me that visual design in CSS is largely about controlling spacing and hierarchy. The "featured" middle plan being taller and more prominent than the others is achieved through a few targeted CSS overrides — once you understand which properties control size and shadow, these effects are surprisingly simple to implement. I also practiced the discipline of starting from a clean starter file and building up incrementally rather than jumping straight to the solution.
